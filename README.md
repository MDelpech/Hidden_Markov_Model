# Hidden Markov Models for Financial Time Series Regime Detection

## Overview

This repository contains the code and report developed for our TER project on **Hidden Markov Models applied to financial time series**.

The objective of the project is to study how Hidden Markov Models can be used to detect **latent regimes** in financial markets, with a particular focus on volatility dynamics through the **VIX index**.

The project combines:

- a theoretical study of Hidden Markov Models,
- an implementation of the main inference algorithms,
- simulation experiments on synthetic data,
- an empirical application to financial volatility data.

The approach is strongly oriented toward **quantitative research**, with an emphasis on statistical estimation, regime-switching models, volatility modelling, numerical stability, and financial interpretation.

---

## Project Motivation

Financial time series often exhibit changes in behaviour over time. Periods of calm markets, transition phases, and stress regimes are not always directly observable from raw data.

Hidden Markov Models provide a natural framework for this type of problem. They assume that the observed time series is generated by an underlying hidden Markov chain, whose states can be interpreted as latent market regimes.

In the context of financial markets, these hidden states may correspond to:

- low-volatility regimes,
- intermediate regimes,
- high-volatility or stress regimes.

This makes HMMs particularly relevant for applications such as volatility modelling, risk management, regime detection, and quantitative trading research.

---

## Objectives

The main objectives of the project are:

1. To present the mathematical framework of Hidden Markov Models.

2. To implement and study the main HMM algorithms:

   - Forward algorithm,
   - Viterbi algorithm,
   - Baum-Welch algorithm.

3. To test the algorithms on simulated data generated from known model parameters.

4. To apply the model to the VIX index in order to detect latent volatility regimes.

5. To discuss the relevance, limitations, and possible extensions of HMMs in quantitative finance.
